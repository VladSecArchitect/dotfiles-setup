# macOS Dotfiles

Конфигурация shell и настройка системы для macOS. Профиль ориентирован на безопасность (AppSec / пентест).

## Структура файлов

| Файл | Назначение |
|------|------------|
| `.bash_profile` | Точка входа для login shell — загружает все остальные dotfiles |
| `.exports` | Переменные окружения (`EDITOR`, `HISTSIZE`, локаль и др.) |
| `.path` | Дополнения к `PATH` — Homebrew, LLVM, тулчейны |
| `.bash_aliases` | Только алиасы оболочки (функций нет) |
| `.functions` | Утилитарные функции оболочки |
| `.bash_prompt` | Промпт PS1/PS2 (тема Solarized, интеграция с git) |
| `.inputrc` | Настройки GNU Readline — биндинги клавиш и автодополнение |
| `.extra.example` | Шаблон для машино-специфичных настроек (не коммитится) |
| `osx_fixes/osx_post_install.sh` | Скрипт настройки macOS — `defaults write`, `pmset`, Homebrew |
| `osx_fixes/Brewfile` | CLI-пакеты Homebrew |
| `osx_fixes/Caskfile` | GUI-приложения Homebrew |
| `setup.sh` | Точка входа для начальной установки на новой машине |

## Установка

```bash
git clone <repo> ~/dotfiles
cd ~/dotfiles
bash setup.sh
```

Или применить только настройки macOS:

```bash
bash osx_fixes/osx_post_install.sh
```
